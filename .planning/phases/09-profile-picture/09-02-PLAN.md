  <files>
    - app/src/main/java/com/jna/tictactoe/screen/profile/ProfileScreen.kt
  </files>
  <action>
    1.  In `ProfileScreen`, wrap the avatar display in a `Box` and add an `IconButton` with an edit icon overlaid on it.
    2.  On icon click, call the ViewModel function to show the image source selection dialog.
    3.  Create a new composable for a bottom sheet or dialog that presents "Camera" and "Gallery" options. Show/hide it based on the ViewModel state.
    4.  Initialize `ActivityResultLauncher` for `PickVisualMedia` (Gallery) and `TakePicture` (Camera).
    5.  When the user selects "Gallery", launch the `pickVisualMedia` launcher.
    6.  When the user selects "Camera", use the Accompanist `rememberPermissionState` to request camera permission. On permission granted, launch the `takePicture` launcher.
    7.  In the `onResult` callback for both launchers, call the `viewModel.onImageSelected(uri)` function.
  </action>
  <verify>

  </verify>

  </done>
